Button

A Button that allows users to take actions, and make choices, with a single tap.

Downside: Since Button props extends the default html button props, Docz will display all props including React.ButtonHTMLAttributes ones 😭

Bug on repo link: https://github.com/doczjs/docz/issues/895

Usage

Available buttons:

<div>
  <Button nature="secondary">Cancel</Button>
</div>
<div>
  <Button nature="primary">Save</Button>
</div>
<div>
  <Button nature="danger">Delete</Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    justifyContent: 'center',
  }}
>
  <Button nature="ghost" size="small">
    <CopyPassword />
  </Button>
</div>

All buttons can display icons, text or mixed content:

<div>
  <Button nature="secondary">
    <Shared /> Share
  </Button>
</div>
<div>
  <Button nature="danger">
    <Shared color="white" /> Delete
  </Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    justifyContent: 'center',
  }}
>
  <Button nature="ghost" size="small">
    <CopyPassword />
  </Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    justifyContent: 'center',
  }}
>
  <Button nature="ghost" size="small">
    Confirm
  </Button>
</div>

The default theme is light (white), but some buttons can be displayed in a Dark theme (Dashlane's dark green) ( ⚠️ danger button is not supported in dark theme):

<div>
  <Button nature="secondary" theme="dark">
    Cancel
  </Button>
</div>
<div>
  <Button nature="primary" theme="dark">
    Save
  </Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    justifyContent: 'center',
  }}
>
  <Button nature="ghost" size="small" theme="dark">
    <CopyPassword color="white" />
  </Button>
</div>

Buttons can be have different sizes [small, medium, large] with medium being the default value if none is specified:

<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    alignItems: 'center',
    justifyContent: 'space-between',
  }}
>
  <Button nature="secondary" size="small">
    Cancel
  </Button>
  <Button nature="secondary" size="medium">
    Cancel
  </Button>
  <Button nature="secondary" size="large">
    Cancel
  </Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    alignItems: 'center',
    justifyContent: 'space-between',
  }}
>
  <Button nature="primary" size="small">
    Save
  </Button>
  <Button nature="primary" size="medium">
    Save
  </Button>
  <Button nature="primary" size="large">
    Save
  </Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    alignItems: 'center',
    justifyContent: 'space-between',
  }}
>
  <Button nature="danger" size="small">
    Delete
  </Button>
  <Button nature="danger">Delete</Button>
  <Button nature="danger" size="large">
    Delete
  </Button>
</div>
<div
  style={{
    display: 'flex',
    flexDirection: 'column',
    alignItems: 'center',
    justifyContent: 'space-between',
  }}
>
  <Button nature="ghost" size="small">
    Decline
  </Button>
  <Button nature="ghost">Decline</Button>
  <Button nature="ghost" size="large">
    Decline
  </Button>
</div>

In disabled mode, buttons will behave as following:

<div
  style={{
    display: 'flex',
    justifyContent: 'space-around',
    width: '100%',
    margin: '10px 0',
  }}
>
  <div>
    <Button nature="secondary" disabled>
      Cancel
    </Button>
  </div>
  <div>
    <Button nature="primary" disabled>
      Save
    </Button>
  </div>
  <div>
    <Button nature="danger" disabled>
      Delete
    </Button>
  </div>
  <div
    style={{
      display: 'flex',
      flexDirection: 'column',
      justifyContent: 'center',
    }}
  >
    <Button nature="ghost" size="medium" disabled>
      Decline
    </Button>
  </div>
  <div
    style={{
      display: 'flex',
      flexDirection: 'column',
      justifyContent: 'center',
    }}
  >
    <Button nature="ghost" disabled>
      <CopyPassword />
    </Button>
  </div>
</div>
